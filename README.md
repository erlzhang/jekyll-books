# Jekyll Books Generator

## Installation

Add this line to your Jekyll site's `Gemfile`:

```ruby
gem "jekyll-books"
```

Add `jekyll-books` to your Jekyll site's `_config.yml`:

```yaml
plugins:
  - jekyll-books
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-books
